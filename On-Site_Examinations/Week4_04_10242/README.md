### 題目：[UVa 10242 - Fourth Point !!](https://onlinejudge.org/index.php?option=com_onlinejudge&Itemid=8&category=14&page=show_problem&problem=1183)

#### 解題思路
這題給定平行四邊形（Parallelogram）的兩條相鄰邊（由四個點構成，其中兩點是重複的），要求出第四個點的座標。

#### 關鍵邏輯：
1. 尋找交點 (Find the Common Point)：題目給了四個座標（共八個數值），其中有一個點是兩條邊的「共同交點」。我們必須先找出這個點。
2. 向量原理 (Vector Logic)：
- 假設三點分別為 $A, B, C$，其中 $B$ 是兩邊的交點（即 $BA$ 和 $BC$ 是兩條邊）。
- 平行四邊形的第四個點 $D$ 可以透過向量加法求得：$\vec{BD} = \vec{BA} + \vec{BC}$。
- 轉換成座標公式：$D = A + C - B$。
3. 座標運算：
- 找出重複點（假設為 $P_{common}$）後，將另外兩個不重複的點相加，再減去這個重複點，即可得到第四點。

#### 模擬運算：
- 輸入：點1 $(x1, y1)$、點2 $(x2, y2)$、點3 $(x3, y3)$、點4 $(x4, y4)$。
- 輸出：保留三位小數。
- Time Complexity (時間複雜度)：$O(1)$。每次運算僅包含簡單的加減法。
- Space Complexity (空間複雜度)：$O(1)$。僅使用固定的 double 變數。

#### 語言差異與踩坑紀錄
* **學到了什麼：**
    - 精確度處理：幾何題目通常要求小數點輸出，使用 double 搭配 printf("%.3f") 是最穩定的做法。
    - 輸入處理：這題的輸入是一行八個數字，直接用 while(cin >> ...) 可以自動處理換行與空格。
    - 幾何對稱性：理解平行四邊形的對角線平分特性或向量加法，可以大幅簡化程式碼，不需要用到複雜的三角函數。

#### 總結 (Takeaway)
- 平行四邊形找第四點 SOP：
    1. 找出重複出現的那個座標（交點）。
    2. 將其他兩個不同的座標相加。
    3. 減去交點座標即為答案。
- 這種「找規律與幾何特性」的方法，比建立直線方程式求交點要快得多。
