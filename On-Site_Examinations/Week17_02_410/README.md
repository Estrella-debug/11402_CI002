### 題目：[UVa 410 – Station Balance](https://onlinejudge.org/external/4/410.pdf)

#### 解題思路
給定 $C$ 個腔室與 $S$ 個標本。每個腔室最多放 2 個標本。為了湊齊 $2C$ 個槽位，不足的標本處視為質量 0
- 目標：分配標本，使得所有腔室質量與平均值的絕對差之和（不平衡度）最小
- 核心特性：為了平衡質量，我們應將「最重」與「最輕」的標本配對，次重與次輕配對，依此類推
- 核心策略：貪心法。先將標本按質量排序，並補零至 $2C$ 個，接著利用「頭尾指針」進行配對

#### 關鍵邏輯：
1. 數據補齊與排序：
    - 標本數 $S$ 若小於 $2C$，需補入 $2C - S$ 個質量為 0 的標本
    - 將所有質量數值由小到大排序
2. 頭尾配對策略：
    - 設定指針 i 從頭 (0) 開始，n-1-i 從尾 (2C-1) 開始
    - 每一組配對 (a[i], a[n-1-i]) 即為一個腔室的分配方案
3. 計算不平衡度：
    - 不平衡度公式為每個腔室的質量總和與平均質量 avg 的絕對差值之總和

#### 模擬運算($C=2, S=3$, 質量 {1, 3, 4}為例)：
1. 補零與排序：補一個 0，質量陣列變為 {0, 1, 3, 4} ($n=4$ 個槽位)
2. 配對：
    - 腔室 0：配對 a[0]=0 與 a[3]=4，和為 4
    - 腔室 1：配對 a[1]=1 與 a[2]=3，和為 4
3. 計算：平均值 avg = (0+1+3+4)/2 = 4
    - 不平衡度：|4-4| + |4-4| = 0
- Time Complexity：$O(S \log S)$，主要耗時在排序
- Space Complexity：$O(C)$，用於儲存腔室與標本分配

#### 語言差異與踩坑紀錄
* **學到了什麼：**
    - 貪心法的應用：將大數與小數配對是解決「總和平衡問題」的最優解，因為這能最有效地中和離群值 (Outliers)
    - 補位技巧：在處理分配問題時，將空位視為「質量為 0 的物體」，能將問題統一化，簡化判斷邏輯
    - 格式化輸出：UVa 題目對輸出格式要求嚴格（如案例後的空行），使用 printf 或調整 cout 格式是避免 Presentation Error 的基本功

#### 總結 (Takeaway)
- 平衡劃分分配 SOP：
    1. 將所有項目質量排序，並補 0 至 $2C$ 個
    2. 將陣列頭尾依序配對
    3. 計算各組和值與平均值的絕對偏差
    4. 累加所有偏差即為最小不平衡度
